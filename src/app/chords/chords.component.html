Color Guide

WHITE = 1st Finger

BLUE = 2nd Finger

GREEN = 3rd Finger

YELLOW = 4th Finger

The suggested finger to use is also indicated as a darkened number (4th finger is 4)

The Notes

'R' ... This indicates the Root note (will be 'C' if the chord is C (major|minor|dominant))

Roman Numerals ... These indicate the progression number (III is the 3rd, IV the 4th, V the 5th, etc)..

Flattened Roman Numerals ... Those which are flattened have a 'b' appending them (e.g. bIII is a flattened 3rd)
